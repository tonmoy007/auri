# 🕯️ Auri — Implementation Plan

> **For Hermes:** A mobile app where teammates anonymously record "confessions" (feedback, gripes, appreciations, ideas) via an AI-driven STT/TTS agent with voice modulation for anonymity, inside a stunning interactive 3D confessional booth. Start/End recording, AI processes it, voice-changed audio forwarded anonymously or deleted.

**Goal:** Build **Auri** — an anonymous AI-driven confession booth mobile app with immersive 3D UI, STT/TTS agent, real-time voice modulation, and forward-or-delete action flow.

**Architecture:** React Native (Expo) mobile app with React Three Fiber for the 3D confessional booth environment. Backend FastAPI server handles STT (Whisper), AI processing (LLM), TTS, voice conversion (RVC/SoX), and anonymous message forwarding. All PII and original voice stripped server-side — the app itself tracks nothing.

**Tech Stack:**
| Layer | Choice |
|-------|--------|
| Mobile | React Native + Expo |
| 3D UI | React Three Fiber + drei + Three.js |
| Backend | FastAPI + WebSocket (for streaming) |
| STT | OpenAI Whisper (or local faster-whisper) |
| TTS | Edge-TTS (free, beautiful voices) |
| **Voice Change** | **SoX audio processing (pitch/formant) + RVC (AI voice conversion)** |
| AI Agent | Any LLM (GPT-4o / Claude) via API |
| DB | PostgreSQL + pgcrypto (PII-safe) |
| Auth | Anonymous tokens only (no identity stored) |

## UX Flow

```
[Tap "Enter Auri"]
    ↓
[3D Booth opens — dim candlelit interior, rotating camera]
    ↓
[AI voice greets you warmly, asks what's on your mind]
    ↓
[Pick Voice Mask — Warm | Robotic | Ethereal | Deep | Random]
    ↓
[User speaks — voice is modulated in real-time, STT streams, booth glows with voice waves]
    ↓
[End Recording — AI processes, categorizes, de-identifies]
    ↓
[Shows transcription ↔ AI summary side by side]
    ↓
[Anonymity choice: Fully Blind | "Someone in your team" context]
    ↓
[Three options: Send Anonymously (with voice-masked audio) | Forward to Someone | Delete]
    ↓
[If Forward: AI suggests best dept/person from manual list]
    ↓
[Moderation queue — designated moderator reviews flagged items]
    ↓
[Delivery via Telegram bot → recipient gets anonymous message + voice-masked audio clip]
    ↓
[Confirmation — booth fades, candle extinguishes]
```

## Data Privacy Design

```
┌─────────────────────────────────────────────────┐
│                 Privacy Layer                     │
├─────────────────────────────────────────────────┤
│ • No user accounts — anonymous device tokens     │
│ • Audio deleted from server after transcription  │
│ • Transcript encrypted at rest (pgcrypto)        │
│ • LLM prompt strips all PII before DB write      │
| • Recipient sees only: "A teammate shared: ..."  │
| • Forwarded messages routed through blind relay  │
| • Optional: self-destruct after 24h read         │
| • **Dual anonymity mode:**                        │
|   → Fully blind (no context at all)              │
|   → "Someone in your team" (team context shown)  │
|   → User picks at send-time                      │
└─────────────────────────────────────────────────┘
```

## ✅ Resolved Decisions

| Question | Decision |
|----------|----------|
| **Recipient discovery** | Manual department list — admin-managed in-app (no LDAP) |
| **Anonymity spectrum** | **Both** — user chooses at send-time: fully blind OR "someone in your team" context |
| **Notification channel** | **Telegram bot** — each confession delivered via anonymous bot message |
| **Moderation** | Yes — flagged content queued to **designated moderator user(s)** via Telegram for review |
| **Environment variety** | Classic booth + optional forest glade + rooftop night |
