# рҹ•ҜпёҸ Auri вҖ” Implementation Plan

> **For Hermes:** A mobile app where teammates anonymously record "confessions" (feedback, gripes, appreciations, ideas) via an AI-driven STT/TTS agent with voice modulation for anonymity, inside a stunning interactive 3D confessional booth. Start/End recording, AI processes it, voice-changed audio forwarded anonymously or deleted.

**Goal:** Build **Auri** вҖ” an anonymous AI-driven confession booth mobile app with immersive 3D UI, STT/TTS agent, real-time voice modulation, and forward-or-delete action flow.

**Architecture:** React Native (Expo) mobile app with React Three Fiber for the 3D confessional booth environment. Backend FastAPI server handles STT (Whisper), AI processing (LLM), TTS, voice conversion (RVC/SoX), and anonymous message forwarding. All PII and original voice stripped server-side вҖ” the app itself tracks nothing.

**Tech Stack:**
| Layer | Choice |
|-------|--------|
| Mobile | React Native + Expo |
| 3D UI | React Three Fiber + drei + Three.js |
| Backend | FastAPI + WebSocket (for streaming) |
| STT | OpenAI Whisper (or local faster-whisper) |
| TTS | Edge-TTS (free, beautiful voices) |
| **Voice Change** | **SoX audio processing (pitch/formant) + RVC (AI voice conversion)** |
| AI Agent | Any LLM (GPT-4o / Claude) via API |
| DB | PostgreSQL + pgcrypto (PII-safe) |
| Auth | Anonymous tokens only (no identity stored) |

## вң… Resolved Decisions

| Question | Decision |
|----------|----------|
| **Recipient discovery** | Manual department list вҖ” admin-managed in-app (no LDAP) |
| **Anonymity spectrum** | **Both** вҖ” user chooses at send-time: fully blind OR "someone in your team" context |
| **Notification channel** | **Telegram bot** вҖ” each confession delivered via anonymous bot message |
| **Moderation** | Yes вҖ” flagged content queued to **designated moderator user(s)** via Telegram for review |
| **Environment variety** | Classic booth + optional forest glade + rooftop night |
